The following files contain the material used in the report:
    - Statistics and Graphs.ipynb
    - Code.ipynb
    - Functions.py

Statistics and Graphs.ipynb contains all the graphs and data manipulation used in the descriptive section of the report.

Code.ipynb contains all the calculations used in the report. The code calculations are presented in the chronological order as presented in the report.

Functions.py contains all the helper functions we have created.
